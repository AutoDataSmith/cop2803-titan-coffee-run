# Titan Coffee Run – COP2803
This project is a front-end web application for Titan Coffee Run, developed as part of St. Petersburg College's COP2803 (Client-side JavaScript) course.
The application is built incrementally across multiple assignments using Agile development principles and AI-assisted programming.
## Features
- Homepage with responsive layout
- Banner slideshow (JavaScript)
- Credit application form with validation
- User registration and login system
- Admin sales dashboard
- Shopping cart and order system
- REST API integration (JSON Server)
- Security vulnerability assessment

## How to Run
1. Clone or download the repository
2. Open the project folder in VS Code
3. Open index.html in a browser
   OR use Live Server in VS Code   
 ## Development Approach
This project follows an Agile, sprint-based approach. Each assignment represents a sprint where new features are added to the application.
AI tools are used as a development partner to:
- Generate and review code
- Explain concepts
- Assist with debugging
- Improve code quality and documentation
